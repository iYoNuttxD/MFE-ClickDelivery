### Elements:
1. **Cart Icon (ðŸ›’)**: Shopping cart SVG icon
   - Color: Gray (default), Primary (on hover)
   - Size: 24x24px (h-6 w-6)
   - Position: Between Admin button and user name

2. **Badge (Number)**:
   - Position: Top-right corner of cart icon
   - Background: Primary-600 (brand color)
   - Text: White, bold, 12px
   - Shape: Circular (20x20px)
   - Content: Number of items in cart
   - Visibility: Only appears when cart has items (count > 0)

3. **Behavior**:
   - Hover: Cart icon changes to primary color
   - Click: Navigates to `/customer/cart`
   - Tooltip: "Meu Carrinho" on hover

### Conditional Rendering:
- **Visible**: Only when user is authenticated AND has 'customer' role
- **Hidden**: For non-customers (restaurant, courier, owner, admin) and unauthenticated users

### Real-time Updates:
- Badge updates automatically when items are added/removed
- Uses `useCartStore.getItemCount()` for live count
- No page reload required
